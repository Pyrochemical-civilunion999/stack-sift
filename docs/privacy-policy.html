Stack Sift — Privacy Policy

Last updated: March 2026

Stack Sift is a browser extension that helps you triage issues on sentry.io pages. This policy describes how the extension handles information.

Data processing

Classification, machine learning, and feedback features run entirely on your device. The extension does not send your error data, stack traces, or feedback to servers operated by the extension author.

What is stored locally

  • chrome.storage.local — optional feedback you save (corrected categories and derived fingerprints) and editor settings (project path, editor choice) you configure.
  • IndexedDB (via TensorFlow.js) — the optional on-device neural network model after you train it in the extension.

This data stays on your computer unless you export it yourself (e.g. JSON export in the UI).

Permissions

  • Host access to sentry.io — required to read the issue page and show the sidebar only on those sites.
  • storage — required to persist feedback, settings, and related state locally.

Third parties

When you use Stack Sift, you are already using Sentry in the browser; their privacy terms apply to sentry.io. The extension does not add remote analytics or advertising trackers.

Contact

For privacy questions, open an issue at github.com/EduardoDePatta/stack-sift.
